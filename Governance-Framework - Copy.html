5. Frequently Asked Questions

Q: Is this framework compatible with MacOS/Linux (Bash/Zsh)?
A: Currently, this version is PowerShell (Windows) optimized. However, the principles (WORKLOGs, .aiignore, binary decision trees) are universal. A Bash port is on the roadmap.
Q: Does this send my code to the cloud?
A: No. The framework is local-first. It organizes your local files (`.ai` folder). Data is only sent if you explicitly use a cloud-based CLI agent (like Gemini or OpenAI), and even then, the framework helps redact PII via the `PLAN` phase.
Q: Why not just use the AI's built-in memory?
A: Context Window limits and Hallucinations. AI memory is ephemeral. A `WORKLOG.md` file is persistent, grep-able, and legally admissible as an audit trail. It bridges the gap between sessions.
