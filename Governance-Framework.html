2. Detailed MOP: The Forensic Method of Procedure

In telecom, we never touch a switch without a MOP. This framework applies that rigor to AI initialization.

PHASE 1

Pre-Deployment Verification

Forensic Principle: Never trust, always verify. Check for active malicious scripts before installation.

$ # Safety Check: No active IEX scripts
$ Get-Process | Where-Object {$_.CommandLine -like "*iex*"} | Stop-Process -Force
$ # Validate PowerShell Execution Policy
$ if ((Get-ExecutionPolicy) -eq "Restricted") { Set-ExecutionPolicy RemoteSigned -Scope CurrentUser }
PHASE 2

Trust Protocol: Installation Without IEX

Security Principle: Hash verification before execution prevents Man-in-the-Middle attacks. We reject `curl | sh`.
