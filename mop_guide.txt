    <section id="mop-detailed">
        <h2>2. Detailed MOP: The Forensic Method of Procedure</h2>
        <p>In telecom, we never touch a switch without a MOP. This framework applies that rigor to AI initialization.</p>

        <div class="mop-phase">
            <div class="phase-header">
                <span class="phase-number">PHASE 1</span>
                <h3>Pre-Deployment Verification</h3>
            </div>
            <p><strong>Forensic Principle:</strong> Never trust, always verify. Check for active malicious scripts before installation.</p>
            <div class="terminal-simulation">
                <span class="prompt">$</span> # Safety Check: No active IEX scripts<br>
                <span class="prompt">$</span> Get-Process | Where-Object {$_.CommandLine -like "*iex*"} | Stop-Process -Force<br>
                <span class="prompt">$</span> # Validate PowerShell Execution Policy<br>
                <span class="prompt">$</span> if ((Get-ExecutionPolicy) -eq "Restricted") { Set-ExecutionPolicy RemoteSigned -Scope CurrentUser }
            </div>
        </div>

        <div class="mop-phase">
            <div class="phase-header">
                <span class="phase-number">PHASE 2</span>
                <h3>Trust Protocol: Installation Without IEX</h3>
            </div>
            <p><strong>Security Principle:</strong> Hash verification before execution prevents Man-in-the-Middle attacks. We reject `curl | sh`.</p>
            <div class="code-block">
<span class="code-comment"># Verification Logic</span>
<span class="code-keyword">$expectedHash</span> = "e3b0c44298fc1c149afbf4c8996fb92427ae41e4649b934ca495991b7852b855"
<span class="code-keyword">$content</span> = <span class="code-keyword">Invoke-WebRequest</span> -Uri $url
<span class="code-keyword">$actualHash</span> = (<span class="code-keyword">Get-FileHash</span> -InputStream $content.RawContentStream).Hash

<span class="code-keyword">if</span> ($actualHash -eq $expectedHash) {
    <span class="code-keyword">.</span> { $content.Content } <span class="code-comment"># Safe execution</span>
} <span class="code-keyword">else</span> {
    <span class="code-keyword">Write-Error</span> "Hash mismatch. Aborting."
}
            </div>
        </div>

        <div class="mop-phase">
            <div class="phase-header">
                <span class="phase-number">PHASE 3</span>
                <h3>Profile Injection: Idempotent & Safe</h3>
            </div>
            <p><strong>Idempotence Principle:</strong> The script checks for markers before modifying the profile, ensuring no duplication or corruption.</p>
            <div class="code-block">
<span class="code-keyword">$Marker</span> = "# --- UNIVERSAL AI ENVIRONMENT ---"
<span class="code-keyword">if</span> ($currentContent -notmatch [regex]::Escape($Marker)) {
    <span class="code-comment"># Create backup first</span>
    <span class="code-keyword">Copy-Item</span> $ProfilePath "$ProfilePath.bak"
    <span class="code-keyword">Add-Content</span> $ProfilePath $AliasCode
}
            </div>
        </div>
